# MZM
MZM(Minimalist Zig Matrices)

This is a simple Zig Matrix library, focused on simplicity and control, with a little bit of ergonomics in the middle.

If you would like to use it, please use it for study or personal means.
Highly recommend not using it in production.

As simple as it is, in the moment there is no support for standard/commun Zig package librarys, it has no dependencies either.
In order to use it, go to "src/matrix" and past matrix.zig in your project, and use as a normal .zig file, no setup else from that.

Some usage examples are yet to come, and how to use the chainable methods and the "Mode Enum".
